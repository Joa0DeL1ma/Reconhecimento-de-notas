# 🎵 Music Note Trainer

Um mini jogo de reconhecimento de notas musicais feito com HTML, CSS e JavaScript puro.

A ideia é simples: aparece uma nota (A–G) na tela e você precisa escolher o nome correto correspondente (Dó, Ré, Mi, Fá, Sol, Lá ou Si). O objetivo é treinar agilidade e fixar a associação entre cifra e nome da nota.

## 🚀 Como funciona

- Geração aleatória de notas
- Sistema de múltipla escolha
- Feedback visual ao acertar ou errar
- Contador de acertos e erros
- Interface moderna com animações suaves

---

## 🛠 Tecnologias usadas

- HTML
- CSS (animações e glass effect)
- JavaScript (manipulação de DOM e lógica do jogo)

Sem frameworks.

---

## ▶️ Como rodar

Basta abrir o `index.html` no navegador  
ou usar a extensão Live Server no VS Code.
